# Функциональные возможности инженерного калькулятора

## Основные функции
- Сложение (+)
- Вычитание (−)
- Умножение (×)
- Деление (÷)
- Скобки ( )

## Инженерные функции
- Возведение в степень x², xʸ
- Квадратный корень √x
- Обратное число 1/x
- Модуль числа |x|
- Процент %

## Тригонометрические функции
- sin(x)
- cos(x)
- tan(x)
- Режим вычислений в градусах

## Логарифмические функции
- ln(x) — натуральный логарифм
- log(x) — десятичный логарифм

## Математические константы
- π (число пи)
- e (число Эйлера)

## Построение графиков
- Построение графика функции вида y = f(x)
- Поддержка базовых функций (линейные, степенные, тригонометрические)
- Задание диапазона значений x
- Отображение графика в координатной плоскости

## Дополнительные функции
- Очистка экрана (C)
- Удаление последнего введённого символа
- Поддержка дробных чисел
- Обработка ошибок (деление на ноль, корень из отрицательного числа)
